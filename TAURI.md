# 🚀 Ruvi AI Hologram OS - Tauri Desktop Compilation Guide
### (টাউরি দিয়ে উইন্ডোজ `.exe` এবং অ্যান্ড্রয়েড `.apk` বিল্ড করার নির্দেশিকা)

Ruvi AI Hologram OS কে একটি অত্যন্ত শক্তিশালী ডেক্সটপ অ্যাপ্লিকেশন হিসেবে ডিজাইন করা হয়েছে, যার সাহায্যে উইন্ডোজ (Windows), লিনাক্স (Linux), এবং অ্যান্ড্রয়েড (Android) সিস্টেমে সরাসরি ফুল অ্যাডমিনিস্ট্রেশন ও সিস্টেম লেভেল কন্ট্রোল (ভলিউম, ব্রাইটনেস, স্ক্রিনশট, পাওয়ার অফ, কমান্ড প্রম্পট এক্সিকিউশন) চালানো সম্ভব। 

এই ডকুমেন্টটিতে টাউরি (Tauri v2) ব্যবহার করে কীভাবে অটোমেটিক ডেক্সটপ `.exe` অথবা অ্যান্ড্রয়েড `.apk` বিল্ড করবেন তার বিস্তারিত বিবরণ দেওয়া হলো।

---

## 🛠️ Prerequisites (প্রয়োজনীয় রিকোয়ারমেন্টস)

টাউরি দিয়ে বিল্ড করার জন্য আপনার কম্পিউটারে **Rust (লার্স্ট)** এবং প্রয়োজনীয় সিস্টেম টুলস ইনস্টল থাকতে হবে। নিচে প্রতিটি প্ল্যাটফর্মের নির্দেশনা দেওয়া হলো:

### 1. Windows (উইন্ডোজের জন্য)
*   **Rust (কার্গো সহ):** [rustup.rs](https://rustup.rs/) থেকে Rust ইনস্টল করুন। ইনস্টল করার সময় ডিফল্ট সেটিংস (Option 1) সিলেক্ট করুন।
*   **C++ Build Tools:** Visual Studio Installer ডাউনলোড করে **"Desktop development with C++"** ওয়ার্কলোডটি ইনস্টল করুন।
*   **WebView2:** উইন্ডোজ ১১ এ এটি বিল্ট-ইন থাকে। উইন্ডোজ ১০ হলে মাইক্রোসফট সাইট থেকে WebView2 Runtime ইনস্টল করে নিন।

### 2. Linux (লিনাক্সের জন্য - Ubuntu/Debian)
টার্মিনালে নিচের কমান্ডটি রান করে ডিপেন্ডেন্সিগুলো ইনস্টল করুন:
```bash
sudo apt update
sudo apt install -y libwebkit2gtk-4.1-dev build-essential curl wget file libssl-dev libgtk-3-dev libayumu-dev ayumu
```
তারপর Rust ইনস্টল করুন:
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### 3. Android (অ্যান্ড্রয়েডের জন্য)
অ্যান্ড্রয়েড অ্যাপ বিল্ড করতে চাইলে আপনার কম্পিউটারে নিচের জিনিসগুলো থাকতে হবে:
*   **Android Studio** এবং **Android SDK**
*   **NDK** এবং **CMake** (Android Studio-র SDK Manager > SDK Tools থেকে ইনস্টল করুন)
*   **Rust Android Targets:** টার্মিনালে নিচের কমান্ডটি রান করুন:
    ```bash
    rustup target add aarch64-linux-android armv7-linux-androideabi i686-linux-android x86_64-linux-android
    ```

## 🏗️ How to Build (কীভাবে বিল্ড করবেন)

আপনার পিসিতে সোর্স কোডটি ক্লোন/ডাউনলোড করার পর নিচের ধাপগুলো অনুসরণ করুন:

### ১. ডিপেন্ডেন্সি ইনস্টল করুন (Install JS Dependencies)
```bash
npm install
```

### ২. টেম্পোরারি আইকন জেনারেট করুন (Generate App Icons)
আমরা অলরেডি একটি চমৎকার স্ক্রিপ্ট অ্যাড করেছি যা আপনার জন্য অটোমেটিক প্লেসহোল্ডার আইকন তৈরি করবে:
```bash
node src-tauri/generate_placeholders.js
```
*(পরবর্তীতে আপনার নিজের কোনো লোগো `src-tauri/icons` ফোল্ডারে রিপ্লেস করে নিতে পারেন)*

### ৩. লোকাল ডেভেলপমেন্ট মোড (Run Developer Deck)
লোকালি টাউরি ডেক্সটপ উইন্ডো ওপেন করে টেস্ট করতে চাইলে নিচের কমান্ডটি দিন:
```bash
npm run tauri dev
```
এটি আপনার লোকাল রিঅ্যাক্ট অ্যাপ এবং এক্সপ্রেস ব্যাকএন্ডকে টাউরি ডেক্সটপ উইন্ডোর ভেতরে লোড করবে।

### ৪. অটোমেটিক `.exe` বা প্রোডাকশন ডেক্সটপ বিল্ড (Build Windows Executable)
সরাসরি সিঙ্গেল উইন্ডোজ `.exe` বা লিনাক্স ইন্সটলার তৈরি করতে নিচের কমান্ডটি রান করুন:
```bash
npm run tauri build
```
*   **Windows output:** `src-tauri/target/release/bundle/msi/` এবং `src-tauri/target/release/` ডিরেক্টরিতে আপনার রেডিমেড `.exe` ফাইলটি পাবেন।
*   **Linux output:** `src-tauri/target/release/bundle/deb/` অথবা `appimage/` ফোল্ডারে বিল্ড ফাইল জেনারেট হবে।

### ৫. অটোমেটিক অ্যান্ড্রয়েড বিল্ড (Build Android .apk)
মোবাইলে চালানোর জন্য অ্যান্ড্রয়েড প্যাকেজ বিল্ড করতে এই কমান্ডটি ব্যবহার করুন:
```bash
npm run tauri android build
```
এটি রান করলে আপনার জন্য রেডিমেড `.apk` ফাইল জেনারেট হবে যা সরাসরি যেকোনো অ্যান্ড্রয়েড ফোনে ইনস্টল করতে পারবেন।

## 🧬 Ruvi Admin Companion Architecture
Ruvi AI Hologram OS-এর ডেক্সটপ কন্ট্রোল প্যানেলটি অত্যন্ত আধুনিক। এটি দুটি উপায়ে কাজ করে:

1.  **Direct Local PowerShell Control:** যখন আপনি আপনার নিজের উইন্ডোজ পিসিতে সোর্স কোডটি রান করবেন, তখন এটি সরাসরি উইন্ডোজের ইন্টারনাল PowerShell API ব্যবহার করে ভলিউম সেট, ব্রাইটনেস, পিসি লক, স্লিপ, শাটডাউন এবং ডেক্সটপ স্ক্রিনশট নিতে পারবে।
2.  **Web-Socket Companion Bridge:** আপনি চাইলে এই রিঅ্যাক্ট ক্লায়েন্টটিকে ক্লাউডে হোস্ট করে আপনার লোকাল পিসিতে একটি ব্যাকগ্রাউন্ড উইন্ডোজ/লিনাক্স কমপ্যানিয়ন এজেন্ট স্ক্রিপ্ট চালু রাখতে পারেন। ক্লাউড রিঅ্যাক্ট ক্লায়েন্টটি লোকাল কমপ্যানিয়নের সাথে সুরক্ষিত WebSocket দিয়ে কানেক্ট হয়ে পিসিকে রিমোটলি কন্ট্রোল করতে পারবে।

## 🎨 Customize App Details (কনফিগারেশন কাস্টমাইজেশন)

Your desktop app name, window size, or bundle identifier can be configured inside `/src-tauri/tauri.conf.json`:

```json
{
  "productName": "Ruvi AI Hologram OS", // Your app name
  "version": "1.0.0",
  "identifier": "com.ruvi.aihologram.os", // Unique bundle identifier
  "app": {
    "windows": [
      {
        "title": "Ruvi AI Hologram OS Admin Deck", // Window Title Bar
        "width": 1280,
        "height": 800,
        "resizable": true
      }
    ]
  }
}
```

## 🤖 GitHub Actions: Auto Release `.exe` (গিটহাব অ্যাকশনস দিয়ে অটোমেটিক রিলিজ)

আমরা আপনার প্রজেক্টে একটি শক্তিশালী **GitHub Actions Workflow** যোগ করেছি। এটি গিটহাবে কোড পুশ করার সাথে সাথে অটোমেটিক ব্যাকগ্রাউন্ডে উইন্ডোজ `.exe` অ্যাপ্লিকেশান তৈরি করে আপনার গিটহাব রিপোজিটরিতে **Draft Release** হিসেবে রেডি করে রাখবে!

### এটি কীভাবে কাজ করে ও কীভাবে ট্রিগার করবেন?

১. **ট্যাগ পুশ করার মাধ্যমে (Recommended):**
   যখনই আপনি আপনার প্রজেক্টের একটি নতুন ভার্সন রিলিজ করতে চাইবেন, তখন একটি নতুন গিট ট্যাগ (যেমন: `v1.0.0`) তৈরি করে পুশ করুন:
   ```bash
   git tag v1.0.0
   git push origin v1.0.0
   ```
   এটি গিটহাবে পুশ হওয়ার সাথে সাথে আপনার রানার অটোমেটিক উইন্ডোজ `.exe` বিল্ড করা শুরু করবে এবং বিল্ড শেষে গিটহাব রিলিজ পেজে একটি ড্রাফট হিসেবে আপলোড করে দেবে।

২. **ম্যানুয়াল ট্রিগারের মাধ্যমে (Manual Dispatch):**
   *   আপনার GitHub Repository-তে যান।
   *   **Actions** ট্যাবে ক্লিক করুন।
   *   বাম পাশের লিস্ট থেকে **"Tauri Release Automation"** সিলেক্ট করুন।
   *   ডান পাশে **"Run workflow"** বাটনে ক্লিক করে ম্যানুয়ালি বিল্ডটি ট্রিগার করুন।

বিল্ড হয়ে যাওয়ার পর আপনি আপনার গিটহাবের **Releases** সেকশনে সরাসরি ইন্সটলার সহ রেডিমেড `.exe` ডাউনলোড করতে পারবেন!

আপনার কোনো প্রশ্ন বা বিল্ড করতে অসুবিধা হলে নির্দ্বিধায় আমাকে জানান! আমরা আপনার সম্পূর্ণ সিস্টেম অ্যাডমিনিস্ট্রেশন এক্সপেরিয়েন্সকে একদম নিখুঁত করে তুলবো। 🚀
